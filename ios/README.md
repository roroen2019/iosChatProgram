
## 주요기술

- Swift
- SwiftUI
- MVVM
- Realm
- kakao login
- StompClientLib

## 주요화면

1. 채팅가능 유저 리스트 화면 / UserListView
2. 채팅 리스트 화면 / ChatListView
3. 1:1채팅 화면 / ChatView
3. 유저 설정 화면 / MoreView

## 간단한 흐름 설명

- 로그인화면 -> 닉네임 입력화면 -> 메인화면(채팅가능 유저 리스트 화면) -> 유저 선택 -> 채팅 진행

## 중요!

프로젝트의 중요한 목표는 채팅을 구현하는것.
유저검색, 유저추가, 각 유저에 맞는 채팅방 개설 기능은 따로 만들지 않았음.
